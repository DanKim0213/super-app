# 슈퍼앱

슈퍼앱은 프론트엔드 역량 (React, TypeScript, Tailwindcss, Jest, Nextjs)을 키우기 위해 시작한 프로젝트입니다.

## 시작하기

```bash
yarn
yarn dev
```
